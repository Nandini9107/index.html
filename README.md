⚡ Sorting Visualizer

A simple and interactive sorting visualizer built using HTML, CSS, and JavaScript.

🚀 Features
- Generate random bars
- Bubble sort visualization
- Smooth animations
- Color indicators:
  - 🔴 Comparing
  - 🟢 Sorted

🛠️ Tech Stack
- HTML
- CSS
- JavaScript
